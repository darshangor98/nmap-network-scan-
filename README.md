## Overview
This project demonstrates a full network reconnaissance and enumeration assessment using Nmap in a controlled lab environment. The goal was to identify open ports, running services, outdated software, and known vulnerabilities without performing exploitation.

---

## Tools Used
- Kali Linux
- Nmap
- Metasploitable (Target VM)

---

## Methodology
- Full TCP port scan (`-p-`)
- Service and version detection
- OS fingerprinting
- Vulnerability detection using Nmap NSE scripts
- Risk-based categorization of all findings
